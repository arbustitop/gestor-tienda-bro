PROJECT OVERVIEW
--------------------------------------------------------------------------------
A complete store management system for custom printed products (shirts, 
sweatshirts, mugs, etc.) with stock control, sales tracking, and automatic 
material calculation for DTF and sublimation printing.

FEATURES IMPLEMENTED
--------------------------------------------------------------------------------
✅ Product Management (CRUD)
   - Create, read, update, delete products
   - Support for sizes, colors, types
   - Stock management
   - Fabric/vinyl material tracking per product

✅ Sales System
   - Shopping cart interface
   - Client information recording
   - Automatic stock updates on sale
   - Automatic material calculation (meters of fabric/vinyl)
   - Transaction history with details

✅ Search & Filters
   - Search by name, color, size
   - Filter by product type
   - Low stock alerts (≤5 units)
   - Real-time search

✅ Statistics & Analytics
   - Daily sales summary
   - Monthly revenue tracking
   - Top selling products
   - Client statistics
   - Inventory valuation
   - Material usage tracking (DTF/sublimation)
   - 7-day sales trends

✅ CSV Export
   - Export all products
   - Export sales history
   - Export low stock items
   - Export statistics report
   - Export individual sale details
   - Excel/Google Sheets compatible

✅ GUI (CustomTkinter)
   - Modern dark theme
   - Responsive layout
   - Sidebar navigation
   - Dashboard with metrics
   - Modal dialogs for forms
   - Visual stock warnings
   - Intuitive workflows

✅ Database (SQLite)
   - Automatic creation on first run
   - Migration system
   - Safe transactions with rollback

   - Foreign key relationships
   - Timestamps for audit

KEY FEATURES EXPLAINED
--------------------------------------------------------------------------------

1. AUTOMATIC MATERIAL CALCULATION
   - Each product has fabric_meters and vinyl_meters fields
   - DTF printing uses fabric (e.g., 0.5m per shirt)
   - Sublimation uses vinyl (e.g., 0.15m per mug)
   - On sale, materials are calculated: meters × quantity
   - Totals accumulated per sale for inventory control

2. STOCK MANAGEMENT
   - Stock updated automatically on each sale
   - Validation prevents overselling
   - Low stock alerts when ≤5 units
   - Stock can be updated manually
   - Transaction rollback on errors

3. SEARCH & FILTERS
   - Real-time search as you type
   - Search across name, color, size
   - Filter by product type
   - Show only low stock items
   - Reset to show all

4. SALES WORKFLOW
   - Enter client name
   - Add products to cart
   - Specify quantities (with validation)
   - Review totals (money, fabric, vinyl)
   - Complete sale (atomic transaction)
   - Stock automatically updated

5. STATISTICS
   - Dashboard shows today's metrics
   - Monthly summaries
   - Top 10 products by sales
   - Client spending rankings
   - Product type distribution
   - Inventory valuation
   - 7-day sales trend

USAGE INSTRUCTIONS
--------------------------------------------------------------------------------

  2. pip install -r requirements.txt
  3. python main.py

First Time Setup:
  1. Database created automatically
  2. Add first product
  3. Make first sale
  4. View statistics

Daily Workflow:
  1. Check dashboard for overview
  2. Review low stock items
  3. Process sales
  4. Update stock as needed
  5. Export reports for accounting
